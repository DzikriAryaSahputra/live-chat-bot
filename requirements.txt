=======================================================
DAFTAR REQUIREMENTS & TEKNOLOGI (TECH STACK)
BPS LIVE CHAT BOT (SISCA)
=======================================================
Dokumen ini berisi daftar lengkap semua alat (tools), bahasa pemrograman,
dan pustaka (dependencies) yang digunakan untuk membangun proyek ini.

-------------------------------------------------------
1. BAHASA PEMROGRAMAN & ENGINE UTAMA
-------------------------------------------------------
- Node.js (v18+) : Engine utama untuk menjalankan server Backend.
- Python (v3.8+) : Engine untuk menjalankan mesin cerdas Rasa NLU.
- HTML/CSS/JS    : Bahasa standar untuk antarmuka pengguna (Frontend).

-------------------------------------------------------
2. DEPENDENSI BACKEND (NODE.JS / NPM)
Lokasi: /backend/package.json
-------------------------------------------------------
Framework & Jaringan:
- express (^5.2.1)    : Web framework untuk membuat API & Server.
- socket.io (^4.8.3)  : Library untuk komunikasi chat dua arah (Real-time).
- cors (^2.8.6)       : Mengizinkan frontend berkomunikasi dengan backend.
- axios (^1.13.6)     : Klien HTTP untuk menembak API eksternal (Groq/Gemini).

Manajemen Data & Database:
- pg (^8.18.0)        : Driver untuk terhubung ke Database PostgreSQL.
- dotenv (^17.3.1)    : Membaca konfigurasi rahasia dari file .env.
- jsonwebtoken (^9.0.3): Mengamankan sesi login Admin (JWT).
- js-yaml (^4.1.1)    : Membaca dan memanipulasi file pelatihan Rasa (.yml) dari Admin Panel.

Pemrosesan PDF & AI RAG:
- multer (^2.1.1)          : Menangkap file PDF yang diunggah dari Dashboard Admin.
- pdf-parse (^1.1.1)       : Mengekstrak teks murni dari file PDF.
- @xenova/transformers (^2.17.2) : Engine AI Lokal (WebAssembly) untuk mengonversi teks menjadi Vektor (Embedding).
- Model AI (Offline)       : Xenova/all-MiniLM-L6-v2 (Dijalankan lokal via Transformers.js).
- Model LLM (Online)       : Groq (Llama-3.1-8b-instant) sebagai Utama, Gemini (2.0-flash) sebagai Cadangan.

-------------------------------------------------------
3. DEPENDENSI NLU (PYTHON / PIP)
Lokasi: /rasa-bot/requirements.txt
-------------------------------------------------------
- rasa           : Framework utama Natural Language Understanding (Pemahaman Bahasa).
- rasa-sdk       : Action server khusus untuk fungsi custom (Python).
- spacy          : Engine NLP tambahan (opsional) untuk klasifikasi yang lebih dalam.

-------------------------------------------------------
4. DATABASE & PENYIMPANAN
-------------------------------------------------------
- PostgreSQL     : Menyimpan riwayat obrolan (chat_logs) dan sesi user.
- Local JSON     : Menyimpan database vektor (/backend/knowledge_docs/embeddings.json).
- Local File     : Menyimpan raw text PDF di dalam folder /backend/knowledge_docs/.

-------------------------------------------------------
5. FRONTEND & UI/UX
Lokasi: /frontend
-------------------------------------------------------
- Tailwind CSS   : Framework CSS modern (via CDN) untuk styling yang sangat rapi dan responsif.
- Plus Jakarta Sans : Tipografi modern resmi dari Google Fonts.
- Material Symbols  : Paket ikon estetik dari Google.
- SweetAlert2    : (Opsional) Library untuk memunculkan pop-up yang cantik.
- Glassmorphism  : Gaya desain CSS khusus untuk Admin Panel.

-------------------------------------------------------
CARA INSTALASI CEPAT (AUTO-INSTALLER)
-------------------------------------------------------
Jika Anda menggunakan Windows, Anda tidak perlu menginstal library di atas satu per satu.
Cukup jalankan file `install.bat` yang ada di luar folder, dan sistem akan otomatis mengunduh semuanya!
