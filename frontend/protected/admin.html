Buku Panduan (User Manual)

Panduan lengkap operasional Admin Dashboard BIPS.

1. Manajemen Live Chat

Fitur Live Chat digunakan untuk memantau dan membalas pesan warga secara real-time.

  • Pengambilalihan Otomatis: Jika warga mengetik kata kunci seperti "Hubungi Admin" pada jam kerja, bot akan otomatis berhenti membalas dan mengoper sesi ke Admin.
  • Mengakhiri Sesi: Untuk mengakhiri sesi obrolan secara paksa, ketikkan perintah /selesai di kotak pesan lalu kirim. Bot BIPS akan kembali melayani warga tersebut.
  • Indikator Butuh Bantuan: Warga yang meminta admin akan mendapatkan badge merah (Butuh Petugas) dan naik ke urutan teratas daftar obrolan.

2. Knowledge Base NLU (Kecerdasan Inti)

Menu ini adalah otak utama bot. Di sini Anda melatih bot mengenali pertanyaan umum (FAQ).

  • Intent: Nama/kategori topik bahasan (contoh: tanya_jam_kerja). Nama harus disambung dengan garis bawah (underscore).
  • Training Phrases: Pancingan kalimat warga. Usahakan masukkan minimal 5 variasi pertanyaan per Intent agar bot semakin pintar mengenali.
  • Penting: Latih AI! Setelah menambah atau mengubah Intent, Anda WAJIB menekan tombol Latih Ulang AI (Train) di sudut kanan atas layar agar perubahan tersimpan permanen.

3. AI Management LLM (RAG / Otak Kedua)

Generative AI (seperti Groq/Gemini) bertindak sebagai pelapis (fallback). Jika bot NLU tidak menemukan jawaban yang tepat, ia akan membaca dokumen PDF yang Anda unggah untuk menjawab warga.

  • Unggah PDF: Anda dapat mengunggah file publikasi resmi BPS (hanya PDF). Bot akan menyerap teksnya.
  • Format Teks: Pastikan PDF yang diunggah berisi teks yang bisa di-copy-paste, bukan gambar hasil scan. AI tidak bisa membaca gambar.
  • Limit Token: Penggunaan AI dihitung dalam jumlah Token. Statistik harian token dapat dilihat di atas halaman.

4. Pengaturan Sistem & Keamanan

Halaman pengaturan digunakan untuk mengubah profil admin dan menetapkan jadwal operasional Live Chat.

  • Jam Operasional: Tentukan hari dan jam di mana warga diizinkan memanggil Admin. Di luar jam ini, bot akan menolak permintaan warga yang ingin chat dengan Admin.
  • Mode Darurat (Libur): Jika dicentang, layanan Admin akan ditutup total terlepas dari jam operasionalnya.
  • Ganti Password: Pastikan mengingat password baru Anda! Jika Anda kelupaan kata sandi dan terkunci dari sistem, hubungi bagian IT BPS untuk mereset sandi Anda melalui terminal/server secara aman.
